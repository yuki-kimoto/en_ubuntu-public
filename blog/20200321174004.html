Perl以外のプログラミング言語のインストール

Perl以外のプログラミング言語のインストールです。

C言語のインストール

C言語は、gccというコンパイラをインストールすると使えるようになります。

PHPのインストール

PHPをインストールする。 

sudo apt install -y php

Rubyのインストール

Rubyをインストールする。 

sudo apt install -y ruby

Javaのインストール

Javaをインストールする。Open JDKと呼ばれる種類のJavaです。実行環境であるJREと開発ツールキットであるJDKをインストールします。 

sudo apt install -y default-jre

JREをインストールするとJDKもインストールされます。

nodejsのインストール

JavaScriptの実行環境であるnodejsをインストールする。 

sudo apt install -y nodejs

Python 2のインストール

Python 2をインストールする。 

sudo apt install -y python

Python 3のインストール

Python 3をインストールする。 

sudo apt install -y python3

僕の好きなこのプログラミング言語ないよというものありましたら、プルリクエスト歓迎。
