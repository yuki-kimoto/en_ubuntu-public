ライブラリの読み込みパス

Ubuntuのライブラリの読み込みパスについての情報です。

aptでインストールしたパッケージのライブラリの読み込みパス

aptでインストールしたパッケージのライブラリは「/usr/lib/x86_64-linux-gnu」にあります。 

/usr/lib/x86_64-linux-gnu

ライブラリの読み込みパスの確認

ライブラリの読み込みパスを確認するには「/etc/ld.so.conf」の中を見ます。 

cat /etc/ld.so.conf

中身を見てみると、さらにincludeされています。 

include ld.so.conf.d/*.conf

ですので、catコマンドで、すべての読み込みファイルを出力してみましょう。 

$ cat /etc/ld.so.conf.d/*.conf

# libc default configuration
/usr/local/lib
# Multiarch support
/usr/local/lib/x86_64-linux-gnu
/lib/x86_64-linux-gnu
/usr/lib/x86_64-linux-gnu
