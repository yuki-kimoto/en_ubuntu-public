nfs-kernel-serverのインストール - NFSサーバー ネットワークファイル共有

複数サーバー間で、ネットワークファイル共有をするにはNFSサーバーをである「nfs-kernel-server」を、aptコマンドでインストールしましょう。 

sudo apt install -y nfs-kernel-server

NFSは、ネットワークファイルシステムの略です。

NFSクライアントとしてNFSサーバーにマウントする機能を提供するnfs-commonも一緒にインストールされます。

NFSサーバーがインストールできたことの確認

NFSサーバーがインストールできたことをsystemctlコマンドのstatusを使って確認しましょう。 

systemctl status nfs-kernel-server

以下のように表示されればインストールされています。

   Active: active (exited) since Mon 2020-05-18 09:51:35 JST; 2min 47s ago
 Main PID: 20222 (code=exited, status=0/SUCCESS)
    Tasks: 0 (limit: 9479)
   CGroup: /system.slice/nfs-server.service
