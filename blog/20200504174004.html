gccのインストール

gccのインストール方法を解説します。aptコマンドを使って、gccをインストールします。sudoを使って管理者権限で実行します。 

sudo apt install -y gcc

gccの主な用途

gccの主な用途は、C言語のコンパイルです。C言語のソースファイルをコンパイルして、オブジェクトファイルを作成できます。

オブジェクトファイルというのは、一つまたは複数のC言語ファイルに対応する処理を含む機械語になったファイルです。

gccはリンカも含んでいます。リンカ機能で、オブジェクトファイルを結合して、実行ファイル、静的ライブラリ、共有ライブラリを作成できます。

gccは、CPANからC言語ソースファイルを含んでいるXSモジュールをインストールするときに必要です。

C++のコンパイラ g++

PerlのXSモジュールでは、C++が使われていることもあるので、合わせてC++のコンパイラであるg++もインストールしておきましょう。
