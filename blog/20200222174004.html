libxml2-devのインストール - Libxml2のヘッダとライブラリ

libxml2-devのインストールのインストール方法です。aptコマンドでインストールします。Libxml2のヘッダとライブラリがインストールされます。 

sudo apt install -y libxml2-dev

インストールされた、Libxml2のヘッダファイルとライブラリの場所については、ヘッダーファイルの読み込みパスとライブラリの読み込みパスを参考にしてください。
