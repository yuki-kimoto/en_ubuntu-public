libio-socket-ssl-perlパッケージのインストール - PerlのIO::Socket::SSL

PerlでHTTPSアクセスが行えるようにlibio-socket-ssl-perlパッケージをaptコマンドを使って、インストールします。sudoを使って管理者権限で実行します。 

sudo apt install libio-socket-ssl-perl

PerlのIO::Socket::SSLモジュールがインストールされます。Net::SSLeayも、IO::Socket::SSLに依存しているので、インストールされます。

HTTP::Tinyを使ってHTTPSにアクセス

(以下は2020年5月16日段階で、まだ試していません)

PerlのコアモジュールであるHTTP::TinyはIO::Socket::SSLを使ったHTTPSアクセスを透過的にサポートしています。
