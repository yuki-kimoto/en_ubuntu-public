select-editor - デフォルトの編集エディタを変更する

select-editorコマンドでデフォルトの編集エディタを変更できます。 

select-editor

以下のようにエディタの選択画面が出るので、お気に入りのエディタを選択します。 

Select an editor.  To change later, run 'select-editor'.
  1. /bin/nano        <---- easiest
  2. /usr/bin/vim.basic
  3. /usr/bin/vim.tiny
  4. /bin/ed
