cronのインストール - 定期プログラム実行

cronをインストールしましょう。cronは定期プログラム実行を行うための常駐アプリケーションです。デフォルトでインストールされていると思いますが、もしインストールされていない場合は、この手順でインストールしてください。aptコマンドを使います。 

sudo apt install -y cron

cronがインストールされて実行されていることをsystemctlで確認します。 

sudo systemctl status cron

activeと表示されれば、cronがインストールされて実行されています。
