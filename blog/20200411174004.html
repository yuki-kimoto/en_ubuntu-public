libmariadbclient-devのインストール - MariaDBクライアントのヘッダとライブラリ

MariaDBクライアントのヘッダやライブラリを含む「libmariadbclient-dev」をaptコマンドでインストールしましょう。

libmariadbclient-devは、PerlのDBIでMariaDBサーバーに接続するためのDBD::MySQLをインストールするために必要です。 

sudo apt install -y libmariadbclient-dev

インストールされた、MariaDBクライアントのヘッダファイルとライブラリの場所については、ヘッダーファイルの読み込みパスとライブラリの読み込みパスを参考にしてください。
