aptのアップデート - apt update

aptをアップデートするには「apt update」を使用します。sudoを使ってroot権限で実行します。 

sudo apt update

aptが参照するURLなどが変更していたりした場合は「apt install」が失敗する場合があります。

ですので、aptを最初に使い始める場合は「apt update」をしておくと安心です。

「apt install gcc」を実行すると、以下のようなエラーがでましたが「apt update」を実行すると、このエラーがでなくなりました。
