vsftpdのインストール - FTPサーバー

FTPサーバーを使える用にするにはvsftpdをインストールします。 

sudo apt install -y vsftpd

FTPは、ファイル転送プロトコルの略です。クライアント・サーバーで、ファイルを交換するためのプロトコルです。

FTPは、暗号化はされません。暗号化が必要な機密な情報は、SSH上に構築されたSFTPを使って、ファイル転送を行いましょう。

vsftpdがインストールされたことの確認

vsftpdがインストールされたことを確認するためにsystemctlコマンドのstatusを使いましょう。 

systemctl status vsftpd
