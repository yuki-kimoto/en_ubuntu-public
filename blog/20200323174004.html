zipのインストール - ZIP形式のファイルの圧縮・解凍

拡張子がzipのZIP形式で圧縮・解凍を行うにはzipパッケージをaptコマンドを使ってインストールします。sudoで管理者権限で実行します。 

# zipのインストール
sudo apt install -y zip

zipコマンドがインストールされたことの確認

ZIP形式のファイルの圧縮を行うzipコマンドがインストールされたことを確認しましょう。 

zip --help

zipパッケージと一緒に、ZIP形式の解凍を行うunzipパッケージもインストールされます。 

unzip --help

一般的には、Linuxにおいては標準的な圧縮形式は拡張子がgzのGZIP形式です。Linux上で圧縮・解凍をおなう場合は、GZIP形式で圧縮・解凍できるgzipコマンド、gunzipコマンドを使用しましょう。tarコマンドの「-z」オプションを使って、圧縮するも一般的です。これは内部的にはgzipのアルゴリズムを使っています。

Windowsにおけるよく利用されている圧縮形式がZIP形式です。何らかのサービスやWebで提供されているファイル形式がZIPで、ZIPを扱う必要がある場合にzipパッケージをインストールします。
