perl-docパッケージのインストール - perldocコマンド

perldocコマンドを使えるようにperl-docパッケージをaptコマンドを使って、インストールします。sudoを使って管理者権限で実行します。 

sudo apt install -y perl-doc

perldocコマンド

perldocコマンドを使えるようになりました。 

perldoc -f substr
