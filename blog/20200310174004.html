Apacheでmod_proxyモジュールを有効にする

UbuntuのApacheでは、プロキシ機能を提供するmod_proxyモジュールがデフォルトでは有効になっていないので、有効にします。 

sudo a2enmod proxy

設定を反映させるためにはApacheを再起動してください。

Apacheのモジュールを有効・無効にする方法については、以下の記事を参考にしてください。

  • Apache設定ファイルを有効・無効にする - 基本設定・モジュール設定・サイト設定

HTTP・HTTPSをプロキシするには別途mod_proxy_httpが必要になるので注意してください。

mod_proxyが存在しない場合のApacheのエラーメッセージ

mod_proxyが存在しない場合にApacheのエラーメッセージの例。 

Invalid command '<Proxy', perhaps misspelled or defined by a module not included in the server configuration
