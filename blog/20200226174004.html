cronのインストール - 定期プログラム実行

cronをインストールしましょう。cronは定期プログラム実行を行うための常駐アプリケーションです。デフォルトでインストールされていると思いますが、もしインストールされていない場合は、この手順でインストールしてください。 

sudo apt install -y cron

cronがインストールされて実行されていることを確認します。 

sudo systemctl status cron

activeと表示されれば、cronがインストールされて実行されています。
