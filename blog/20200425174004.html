imagemagickのインストール - ImageMagickのconvertコマンド等

ImageMagickのconvertコマンド等を使えるようにimagemagickパッケージをaptコマンドを使って、インストールします。sudoを使って管理者権限で実行します。 

sudo apt install -y imagemagick

ImageMagickのコマンドを実行

ImageMagickのコマンドを使えるようになりました。 

convert --help
