libperl-devのインストール - Perlのヘッダとライブラリ

libperl-devのインストール方法です。Perlのヘッダとライブラリがインストールされます。 

sudo apt install -y libperl-dev

インストールされた、Perlのヘッダファイルとライブラリの場所については、ヘッダーファイルの読み込みパスとライブラリの読み込みパスを参考にしてください。
