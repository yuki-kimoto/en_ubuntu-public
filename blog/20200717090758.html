Apacheのバージョンを調べる - apachectl -v

Apacheのバージョンを調べるには「apachectl -v」を使用します。 

apachectl -v

以下は、Apacheのバージョンの出力例です。 

Server version: Apache/2.4.29 (Ubuntu)
Server built:   2020-03-13T12:26:16
